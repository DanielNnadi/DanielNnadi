The Problem

Supplier on-time delivery was only 87%. Manual tracking took 40 hours per week. No real-time visibility into supplier performance.

Architecture Overview

Supplier Data → Process Mapping → SQL Database → Power BI → Real-time Tracking

The Solution

Led Lean Six Sigma process improvement. Mapped current processes, identified 7 types of waste, eliminated redundant approvals. Built Power BI dashboard for real-time supplier tracking.

Results
  • Supplier on-time delivery: 87% → 98%
  • Weekly manual effort: 40 hours → 14 hours
  • Approvals reduced from 3 people to 1 for orders under $10K
Key Takeaway

40% of manual work was duplicate data entry. Fix the process before training people.
