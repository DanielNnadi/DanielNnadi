The Problem

Defem Autocare needed an inventory platform. Three external vendors proposed 50+ features, but user research showed staff just wanted accurate, consistent data — not more features.

Vendor APIs → API Contracts → SQL Database → Power BI → 95% User Adoption

The Solution

Conducted user research with 50+ staff members. Cut vendor feature list by 30% before development. Defined API contracts and SQL schemas first.

Led SQL architecture redesign and indexing. Delivered 8 Power BI executive dashboards with defined SLAs under 10 seconds.

Results
  • User adoption: 95% within 30 days
  • Vendor management: Cut feature list by 30% based on user research
  • Performance: Queries from 45 seconds to 8 seconds
  • Delivery: On time with clear API contracts
Key Takeaway

User research before technical decisions saved 30% of development work. Data consistency mattered more to users than feature quantity.
