The Problem

No unified threat detection system existed. Security events were scattered across different logs and tools. The team had no visibility into potential threats in real time.

Architecture Overview
📊 User Browser → ⚙️ React Frontend → 💾 Node.js API → ☁️ AWS EC2 → 🎯 Database
User → React UI → REST API → AWS Deployment → Data Storage → Live Dashboard
The Solution

Built a full-stack threat detection platform that aggregates security events, analyzes them in real time, and provides a dashboard for monitoring.

Managed 2 freelance developers across 3 time zones (US, Europe, Nigeria). Deployed on AWS EC2 with CI/CD pipeline using GitHub Actions.

Results
  • Daily API events: 1,000+ handled reliably
  • Team management: Successfully coordinated 2 developers across 3 time zones
  • Deployment: Live at danielnnadi.github.io/Armorai
Key Takeaway

Managing distributed teams requires clear communication protocols and async documentation. Regular check-ins across time zones kept everyone aligned.
